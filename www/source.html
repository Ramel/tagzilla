Source Code

The TagZilla source code is currently licensed under the Mozilla Public License. It can be accessed in any of the following ways:

  • Install TagZilla, then read the source in Mozilla's Chrome directory
  • View the source using the CVS web interface
  • Check out the source using anonymous CVS with the following commands (password is guest): 
    cvs -d :pserver:guest@mozdev.org:/cvs login
cvs -d :pserver:guest@mozdev.org:/cvs co tagzilla
