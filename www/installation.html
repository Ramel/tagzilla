Installation

Note to Linux users

You must be root to install TagZilla and other such Mozilla extensions at this time. (That probably goes for other multi-user OSes too, but I don't know which off the top of my head.)

TagZilla 0.008

Don't let the version number scare you. I've been dogfooding it since I started, so it should be pretty stable. Of course, having said that, you're trying this at your own risk.
